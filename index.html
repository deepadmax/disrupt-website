Twitch Chat controls everything.

Disrupt is an application that lets your Twitch audience affect your gameplay by redeeming channel rewards. It's written to be extremely modular, and the code is open source, so if you're not happy with the easy configuration that's immediately available, you can reprogram it yourself to do exactly what you want.
